# Data Availability

## Eddy Covariance Data

Half-hourly eddy covariance measurements from the Kuibyshev Reservoir tower site are **available upon request** due to ongoing research activities. Please contact the corresponding author for data access.

### Data Description

| Parameter | Description | Units | Temporal Resolution |
|-----------|-------------|-------|---------------------|
| NEE_f | Net Ecosystem Exchange (gap-filled) | µmol CO₂ m⁻² s⁻¹ | 30 min |
| GPP_f | Gross Primary Production (partitioned) | µmol CO₂ m⁻² s⁻¹ | 30 min |
| Reco | Ecosystem Respiration (partitioned) | µmol CO₂ m⁻² s⁻¹ | 30 min |
| CH4_f | Methane flux (gap-filled) | µmol CH₄ m⁻² s⁻¹ | 30 min |
| Tair | Air temperature | °C | 30 min |
| Ustar | Friction velocity | m s⁻¹ | 30 min |
| qc_* | Quality control flags | 0-2 | 30 min |

### Processing

- Gap-filling and flux partitioning performed using **REddyProc** (Wutzler et al., 2018)
- Quality control following ICOS protocols
- Ustar filtering applied for nighttime data

## Satellite Data

All satellite data are freely available from Google Earth Engine:

### Sentinel-2 Surface Reflectance
- **Collection**: `COPERNICUS/S2_SR`
- **Cloud Probability**: `COPERNICUS/S2_CLOUD_PROBABILITY`
- **Bands used**: B2-B12, SCL
- **Spectral indices**: NDVI, NDWI, MNDWI, AWEInsh, MTCI, CIG, EVI2, etc.

### Landsat 8/9 Collection 2 Level-2
- **Collections**: 
  - `LANDSAT/LC08/C02/T1_L2`
  - `LANDSAT/LC09/C02/T1_L2`
- **Product**: Land Surface Temperature (ST_B10)
- **Processing**: Scaled to °C using Collection 2 coefficients

### Sentinel-1 SAR (optional)
- **Collection**: `COPERNICUS/S1_GRD`
- **Polarization**: VV, VH
- **Indices**: RVI, RFDI, DpRVI

## Sample Data

The `sample/` directory contains example data structures to help users understand the required input formats:

- `training_data_sample.csv` - Example structure for GPP/Reco training data
- `ch4_training_sample.csv` - Example structure for CH₄ training data

**Note**: These are synthetic examples for format reference only, not actual measurements.
